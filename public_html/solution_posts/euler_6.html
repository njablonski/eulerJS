6. Sum square difference

Read the problem definition at projecteuler.net

The sum of the squares of the first ten natural numbers is:

12 + 22 + ... + 102 = 385

The square of the sum of the first ten natural numbers is:

(1 + 2 + ... + 10)2 = 552 = 3025

Hence the difference between the sum of the squares of the first ten natural numbers and the square of the sum is 3025 âˆ’ 385 = 2640.

Find the difference between the sum of the squares of the first one hundred natural numbers and the square of the sum.

For this challenge we need one function that sums the numbers in a range and squares the total, and another that squares each number in the range and adds those products together. Our two functions, sumTheSquares() and squareTheSum(), will have similar for loops. Each function loops through the range 1-100, but sumTheSquares() squares the current number before adding to the running sum, and squareTheSum() doesn't. Instead, squareTheSum() adds the unmodified number to its running sum, and then squares the final value of the sum after the loop is finished.

In this solution, our main function takes in a value for the top end of the range, then uses the functions above to calculate the difference between the square of the sums and the sum of the squares of the range from 1 to the given number. That difference is then written to the output area.

When run on the number 100, our code outputs the number 25164150 -- which is correct.
